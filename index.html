What is the modulo operator?
The modulo is a mathematic operation that finds the remainder of a number when it is devided by another number. It is expressed as %.
How to calculate modulo operator?
To find the modulo of a specific number you should find the remainder of a number when it is devided by another number. It is expressed like this x%y.
Example: Find the answer of the following expression:
15 % 2
answer/
15 % 2 = 1
Why use modulo operator?
The modulo operator is used to find out if a specific number is odd or even. It is also used to keep track of the index of the next free spot in a circular array.
